# Personal Streaming Hub A simple, static, self-hosted streaming dashboard that pulls together free & legal content sources into one clean interface. ## Preview ![Streaming Hub Screenshot](screenshot.png) *(Add a `screenshot.png` image in the repository root to display the preview above.)* ## Features - One-click tiles for your favorite free & legal streaming sources (YouTube subscriptions, Internet Archive movies, podcasts, etc.) - Add, edit, and delete custom tiles - Search by title or tags - Import/export your tile collection (JSON) - 100% client-side, no backend required ## Getting Started ### Option 1: Run locally 1. Download the repository. 2. Open `index.html` in your browser. ### Option 2: Host on GitHub Pages 1. Fork or clone this repository. 2. Make sure `index.html` is in the root of your default branch. 3. Go to **Settings → Pages**. 4. Under **Branch**, select `main` (or your default branch) and set the folder to `/ (root)`. 5. Save, then visit your hub at: ``` https://your-username.github.io/your-repo/ ``` ## Adding Links - Click **Add Link** in the dashboard. - Enter the title, URL, and optional tags. - Links are saved to your browser’s local storage. ## Backup & Restore - **Export** saves your tiles as a JSON file. - **Import** restores them on another device or browser. --- Enjoy your personal streaming hub!
